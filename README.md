# 📚 Repositório de Atividades — JavaScript e TypeScript

Este repositório reúne **diversas atividades e exercícios desenvolvidos durante os estudos de JavaScript e TypeScript**, com foco em conceitos importantes de programação.

Os projetos exploram principalmente:

- 🧠 **Programação Orientada a Objetos (POO)**
- 📦 **Modularização de código**
- 🔁 **Manipulação de coleções (Array, Set, etc.)**
- 🏗 **Organização de projetos**
- ⚙️ **Execução de aplicações com Node.js**
- 🧮 **Lógica de programação**

Cada atividade está organizada em sua **própria pasta**, contendo:

- o **código-fonte**
- um **README.md próprio** explicando o funcionamento da atividade
- instruções de execução quando necessário

---

## 📂 Estrutura do repositório

Cada pasta representa **uma atividade ou projeto independente**, permitindo estudar e executar os exemplos separadamente.

---

## 🚀 Objetivo

O objetivo deste repositório é **documentar a evolução nos estudos de programação**, servindo como:

- registro de aprendizado
- prática de organização de projetos
- prática de uso do Git e GitHub

---

## 💻 Tecnologias utilizadas

- JavaScript
- TypeScript
- Node.js
- Git e GitHub
